# Corporate Swiss Knife

Corporate Swiss Knife, profesyonellerin günlük iş akışlarını tek bir merkezden yönetmeleri için geliştirilmiş, gizlilik odaklı ve hepsi bir arada bir verimlilik araç setidir. Modern iş hayatının ihtiyaç duyduğu temel araçları, tamamen çevrimdışı (offline) çalışan sade bir arayüzde birleştirir.

Corporate Swiss Knife is a privacy-focused, all-in-one productivity toolkit designed for professionals to manage their daily workflows from a single hub. It combines essential modern business tools into a minimalist interface that operates entirely offline.

---

## Özellikler / Features

### Türkçe
- Veri Gizliliği: Tüm işlemler tarayıcı tabanlıdır; verileriniz cihazınızdan dışarı çıkmaz.
- Kapsamlı Modüller: Kanban panosu, OCR (görselden metne), Excel formül hata ayıklayıcı ve Pomodoro sayacı.
- Minimalist Tasarım: Plus Jakarta Sans fontu ile modern ve göz yormayan kullanıcı deneyimi.
- Hızlı Erişim: Modüller arası geçiş için optimize edilmiş klavye kısayolları.

### English
- Data Privacy: All operations are browser-based; your data never leaves your device.
- Comprehensive Modules: Kanban board, OCR (image-to-text), Excel formula debugger, and Pomodoro timer.
- Minimalist Design: A modern, distraction-free user experience powered by Plus Jakarta Sans.
- Quick Access: Optimized keyboard shortcuts for seamless navigation between modules.

---

## Araçlar / Tools

- Kanban: Görev yönetimi ve iş takibi. / Task management and workflow tracking.
- OCR: Ekran görüntülerinden anında metin ayıklama. / Instant text extraction from screenshots.
- Excel Debugger: Karmaşık formülleri analiz etme. / Analysis of complex formulas.
- Doc Studio: Hızlı dökümantasyon ve not alma. / Rapid documentation and note-taking.
- Pomodoro: Odaklanma ve mola yönetimi. / Focus and break management.

## Kullanım / Usage

1. Dosyayı indirin veya depoyu klonlayın. / Download the file or clone the repository.
2. "corporate-swiss-knife.html" dosyasını tarayıcınızda açın. / Open "corporate-swiss-knife.html" in your browser.
3. Kurulum gerektirmeden kullanmaya başlayın. / Start using it immediately with no installation required.

## Kısayollar / Shortcuts

- Ctrl + K : Modül Arama / Search Modules
- Ctrl + V : OCR Görüntü Yapıştır / Paste Image in OCR
- Esc : Aramayı Kapat / Close Search
